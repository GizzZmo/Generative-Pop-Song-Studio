# Contributor Covenant Code of Conduct

This project follows the Contributor Covenant Code of Conduct. We expect all contributors to adhere to the standards described below.

Standards
- Be respectful and collaborative.
- No harassment or discriminatory language.
- Respect different perspectives and experiences.
- When in doubt, communicate clearly and politely.

Reporting
- If you experience or witness unacceptable behavior, please contact the maintainers by opening a confidential issue or emailing the project maintainer.

For a full copy of the Contributor Covenant, see: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
