# Ethical use, licensing, and attribution

## Licensing

- Add a LICENSE file (e.g., MIT) at repository root and ensure any third-party assets are compatible.
- Each third-party model or dataset must be accompanied by a license note in docs/ and a NOTICE file as required.

## Model & dataset policy

- Do NOT commit large model binaries to the repo.
- Provide scripts to download models and store checksums.
- Document the source and license for each model/dataset used.

## Ethical guidelines & prohibited uses

- Do not use the project to impersonate real individuals without explicit consent.
- Avoid generating or distributing content that violates copyrights or privacy.
- Maintain transparency around synthesized voices and music (e.g., metadata tags saying “AI-generated”).

## Attribution

- Include required attribution for third-party models per their license.
- When releasing demos or public outputs, include a short statement about model sources and licenses.
